第一步：使用font-face声明字体 
@font-face {font-family: 'iconfont';
    src: url('iconfont-746a55922f.eot'); /* IE9*/
    src: url('iconfont-746a55922f.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('iconfont-64910d2a93.woff') format('woff'), /* chrome、firefox */
    url('iconfont-d8a7c3a6e0.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('iconfont-5eefab63d4.svg#iconfont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用iconfont的样式 
.iconfont{
    font-family:"iconfont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面 
<i class="iconfont">&#x33;</i>
